# Universal Process Memory Weaver

Production-grade universal memory trainer and reverse engineering platform for any Windows 10+ process.

**Features**
- Attach to any process (games, browsers, office, LLMs, etc.)
- Multi-engine AOB scanning (Aho-Corasick, KMP, BMH)
- SeDebugPrivilege + Wow64 handling
- Architecture-specific DLL injection
- Revolutionary reverse engineering suite
- Crash-proof automation and persistence
- Local LLM assistance

## Quick Start
1. Run `install.bat` as Administrator
2. `luajit main.lua`

Full documentation in `docs/`.
